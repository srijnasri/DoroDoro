## About

A simple pomodoro app that helps to maintain focus and increase productivity. Implemented simple, intuitive UI. 

## Functionality

- Start focus timer
- Pause focus timer
- Reset focus timer
- Two modes for work and break
- Background music for focus

## TechStack

Made using swiftUI, AVKit.
